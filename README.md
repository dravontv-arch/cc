# URL Shortener - Сервис сокращения ссылок

## Описание
Современный сервис для сокращения URL ссылок с мощной админ-панелью и подробной аналитикой.

## Требования
- PHP 8.0+
- MySQL 10.5+
- Apache/Nginx

## Установка

### 1. Настройка конфигурации
Откройте файл `config.php` и настройте параметры подключения к базе данных:

```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'url_shortener');
define('DB_USER', 'ваше_имя_пользователя');
define('DB_PASS', 'ваш_пароль');
```

### 2. Создание базы данных
Откройте в браузере: `http://ваш-сайт/install.php`

Скрипт автоматически создаст:
- Базу данных
- Все необходимые таблицы
- Администратора по умолчанию

**Логин:** admin  
**Пароль:** admin123

⚠️ **ВАЖНО:** После установки удалите файл `install.php`!

### 3. Настройка .htaccess (для Apache)
Для работы коротких ссылок добавьте в .htaccess:

```apache
RewriteEngine On
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^([a-zA-Z0-9_-]+)$ redirect.php?code=$1 [L,QSA]
```

### 4. Настройка Nginx
Для Nginx добавьте в конфиг:

```nginx
location / {
    try_files $uri $uri/ /redirect.php?code=$uri;
}
```

## Использование

### Главная страница
- Введите длинную ссылку
- Optionally укажите свой алиас или длину случайного кода
- Нажмите "Сократить"

### Админ-панель
Доступна по адресу: `/admin/`

#### Дашборд
- Общая статистика
- Графики переходов
- Топ ссылок
- Последние добавленные ссылки

#### Управление ссылками
- Просмотр всех ссылок
- Поиск и фильтрация
- Активация/деактивация
- Удаление

#### Аналитика
- Детальная статистика по периодам
- Активность по часам и дням
- Статистика по браузерам, ОС, устройствам
- Топ рефереров
- Геостатистика

#### Настройки
- Изменение длины ссылки по умолчанию
- Включение/отключение пользовательских алиасов
- Смена пароля администратора

#### Логи
- Журнал всех действий администраторов

## API

### POST /api/shorten.php
```json
{
    "url": "https://example.com/long-url",
    "alias": "my-custom",
    "length": 6
}
```

Ответ:
```json
{
    "success": true,
    "message": "Ссылка успешно сокращена",
    "data": {
        "id": 1,
        "original_url": "https://example.com/long-url",
        "short_code": "abc123",
        "short_url": "http://yoursite.com/abc123"
    }
}
```
