# URL Shortener - Сервис сокращения ссылок

## Возможности

✅ **Сокращение ссылок**
- Случайный код настраиваемой длины (4-20 символов)
- Пользовательские алиасы (можно включить/выключить в настройках)
- Все операции только через админ-панель

✅ **Админ-панель с детальной аналитикой**
- Дашборд с общей статистикой
- Управление ссылками: создание, поиск, активация/деактивация, удаление
- Аналитика: переходы по времени, браузерам, ОС, устройствам, реферерам
- Логи действий администратора
- Настройки системы и смена пароля

✅ **Адаптивный дизайн**
- Работает на всех устройствах (Desktop, Tablet, Mobile)

## Установка

### 1. Настройка базы данных

Отредактируйте `config.php`:

```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'url_shortener');
define('DB_USER', 'root');
define('DB_PASS', 'ваш_пароль');
```

### 2. Создание таблиц БД

Запустите в браузере: `http://your-domain.com/install.php`

После успешной установки **удалите файл install.php**!

### 3. Настройка веб-сервера

#### Для Apache:
Файл `.htaccess` уже создан в корне. Убедитесь, что включен mod_rewrite:

```bash
a2enmod rewrite
systemctl restart apache2
```

#### Для Nginx:
Используйте пример конфигурации из `nginx.conf.example`:

```bash
cp nginx.conf.example /etc/nginx/conf.d/url-shortener.conf
# Отредактируйте путь к файлам и домен
systemctl reload nginx
```

### 4. Вход в админ-панель

- URL: `http://your-domain.com/admin/login.php`
- Логин: `admin`
- Пароль: `admin123`

**Важно:** Сразу смените пароль в настройках!

## Использование

### Создание ссылки (только через админку)

1. Войдите в админ-панель
2. Перейдите в раздел "Все ссылки"
3. Заполните форму:
   - Длинная ссылка (обязательно)
   - Свой алиас (необязательно, если разрешено в настройках)
   - Длина случайного кода (4-20 символов)
4. Нажмите "Создать ссылку"

### Доступ к сокращенной ссылке

`http://your-domain.com/Hdn` (где Hdn - ваш короткий код)

## Настройки

В разделе "Настройки" админ-панели можно изменить:

- Длину короткой ссылки по умолчанию
- Разрешение пользовательских алиасов
- Название и описание сайта
- Пароль администратора

## Аналитика

Система собирает подробную статистику:

- Количество переходов (всего, сегодня, вчера, за неделю, за месяц)
- Уникальные IP
- Браузеры и ОС пользователей
- Типы устройств (desktop/mobile)
- Рефереры (откуда пришли)
- География (страны, города) - при наличии данных
- Активность по часам и дням недели

## Требования

- PHP 8.0+
- MySQL 5.7+ / MariaDB 10.2+
- Apache с mod_rewrite или Nginx
- PDO MySQL extension

## Безопасность

- Хэширование паролей (PASSWORD_DEFAULT)
- Prepared statements для защиты от SQL injection
- Логирование всех действий администратора
- Защита служебных файлов (.htaccess/nginx config)
